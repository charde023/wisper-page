## 1. P0 — 코덱스 프록시 클라이언트 `mac/llm.py`

```python
"""코덱스 프록시(localhost:18080) OpenAI-호환 클라이언트. 종량0(구독). claude -p 안 씀."""
from __future__ import annotations
import os, urllib.request, json
from openai import OpenAI   # pip install openai

BASE = os.environ.get("CODEX_PROXY_BASE", "http://127.0.0.1:18080/v1")
_client = OpenAI(base_url=BASE, api_key="codex-proxy-dummy")  # 프록시가 ~/.codex auth 사용

def healthy() -> bool:
    try:
        with urllib.request.urlopen(BASE + "/models", timeout=5) as r:
            return b"gpt-5" in r.read()
    except Exception:
        return False

def chat(model: str, system: str, user: str, temperature: float = 0.2) -> str:
    r = _client.chat.completions.create(
        model=model,
        messages=[{"role": "system", "content": system},
                  {"role": "user", "content": user}],
        temperature=temperature,
    )
    return r.choices[0].message.content
```
설치: `pip install -U openai` (whisper venv에). **게이트**: `python -c "from mac.llm import healthy; print(healthy())"` → True.

## 2. P1 — 교정 `mac/transcript_clean.py` (gpt-5.4-mini)

```python
"""raw 전사 → 문맥교정. youtube.json의 제목·description·링크를 고유명사 힌트로 주입."""
from __future__ import annotations
import json, sys
from pathlib import Path
sys.path.insert(0, str(Path(__file__).resolve().parent))
from llm import chat

SYS = ("You are an expert transcript editor. Fix ASR errors in this English tech-talk "
       "transcript: correct misheard proper nouns, product names, and 'Cloud'→'Claude' "
       "type errors using the provided hints. Fix sentence boundaries and remove filler. "
       "Keep meaning and all content. Output ONLY the cleaned English transcript.")

def main(ws: str) -> int:
    w = Path(ws)
    raw = (w / "transcript.txt").read_text(encoding="utf-8")
    meta = json.loads((w / "youtube.json").read_text(encoding="utf-8"))
    hints = f"Title: {meta.get('title')}\nDescription:\n{meta.get('description','')[:1500]}"
    out = chat("gpt-5.4-mini", SYS, f"HINTS:\n{hints}\n\nTRANSCRIPT:\n{raw}")
    (w / "transcript_clean.txt").write_text(out.strip() + "\n", encoding="utf-8")
    print(f"CLEAN_OK {len(out)} chars -> {w/'transcript_clean.txt'}")
    return 0

if __name__ == "__main__":
    raise SystemExit(main(sys.argv[1]))
```
**게이트**: 6개 중 1개로 실행 → `transcript_clean.txt` 생성, 고유명사 교정 육안 확인(Spotify=Honk/Niklas).
> 긴 전사는 청크 분할 후 이어붙임(토큰 한도). 초안은 단일 호출, 길면 분할 로직 추가.

## 3. P2 — 노트화 `mac/author_note.py` (gpt-5.5)

```python
"""transcript_clean + youtube.json + note_template → 한국어 학습노트 MD → 볼트."""
from __future__ import annotations
import json, re, sys
from pathlib import Path
sys.path.insert(0, str(Path(__file__).resolve().parent.parent))  # workflow/youtube
sys.path.insert(0, str(Path(__file__).resolve().parent))
from yt_lib import load_config
from llm import chat

RULES = ("당신은 차드(비개발자·이커머스 사장)를 위한 한국어 학습노트 작성자다. "
         "note_template.md 골격을 그대로 채운다. 규칙: 이모지 금지, '==하이라이트==' 금지, "
         "강조는 **볼드**와 <span style=\"color:#ef6c00\">…</span> 2종만. 영어 raw가 아닌 "
         "교정본을 한국어로 번역·구조화. 핵심 영어용어 괄호 병기. 접기식 교정 전사 포함. "
         "frontmatter는 메타에서 채움(status: 정리완료, created: 2026-07-02). "
         "출력은 완성된 마크다운 본문만(코드펜스 없이).")

def main(ws: str) -> int:
    w = Path(ws)
    clean = (w / "transcript_clean.txt").read_text(encoding="utf-8")
    meta = json.loads((w / "youtube.json").read_text(encoding="utf-8"))
    tmpl = (Path(__file__).resolve().parent.parent / "note_template.md").read_text(encoding="utf-8")
    user = (f"[템플릿]\n{tmpl}\n\n[메타(youtube.json)]\n{json.dumps(meta, ensure_ascii=False)}\n\n"
            f"[교정 전사]\n{clean}")
    md = chat("gpt-5.5", RULES, user, temperature=0.3)
    title = re.sub(r'[\\/:*?"<>|]', "-", meta.get("title", meta["id"]))
    dest = Path(load_config()["vaultNoteDir"]) / f"{title}.md"
    dest.write_text(md.strip() + "\n", encoding="utf-8")
    print(f"NOTE_OK -> {dest}")
    return 0

if __name__ == "__main__":
    raise SystemExit(main(sys.argv[1]))
```
**게이트**: 1개 노트 생성 → 볼트 규칙(이모지·`==` 0) 검수, 방금 소넷 수동본과 품질 대조.

"""학습노트 MD → HTML → charde023/page 의 study-notes/ 그룹에 발행. 루트 랜딩은 안 건드림."""

## 5. P4 — 오케스트레이터 `mac/nightly.py`

```python
"""감시→큐레이션→전사→교정→노트→인덱스·커밋→발행→알림. 멱등·실패복구."""
# 흐름(요지):
# 0. llm.healthy() 아니면 대기/알림 후 종료
# 1. rss_watch.py --json → 신규 목록 (아직 --mark 안 함)
# 2. curate 느슨 필터: score >= 임계(E1) → 노트대상, 미만 → 전사만(옵션) or 스킵
# 3. run_youtube.py --url ... (페치+전사, 멱등)
# 4. 각 ws: transcript_clean.py (없으면)
# 5. 각 ws(노트대상): author_note.py (볼트에 노트 없으면)
# 6. rebuild_index.py + 볼트 git add/commit/push (reset --soft 방식으로 레이스 회피)
# 7. publish_study_notes.py (신규 노트만) → page push
# 8. 전 단계 성공한 영상만 rss_watch --mark (seen 마킹) → 실패분 다음밤 재개
# 9. notify(요약: 신규 n·노트 n·발행 링크·실패 사유)
```
**게이트**: 미수집 없을 때 무동작 종료, 있을 때 1개 E2E(전사→교정→노트→발행→마킹) 완주.
